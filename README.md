# Spice Garden Restaurant Website UI/UX Case Study

## Project Overview
Spice Garden is a restaurant website designed to help users explore menus, learn about the restaurant, reserve tables, and contact the restaurant easily.

## UX Process
- User Research
- User Persona
- User Journey
- Low Fidelity Wireframes
- Design System
- High Fidelity Designs
- Interactive Prototype

## Pages
- Home
- Menu
- About
- Contact

## Tools Used
- Figma

## Features
- Simple Navigation
- Food Menu Showcase
- Reservation Section
- Customer Reviews
- Contact Information
